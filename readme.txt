An easy way to host a blog theme party.  The plugin requires free signup to the thinktag.co.uk console for you to add/remove participants.

== Description ==

A simple way to host blog theme party on your WordPress blog or website. Multiple themes can be managed from a single console. 
You can have upto 30 participants in each theme. Auto pruning of blogs that do not reciprocate in adding the code on their pages via configuration.

**Prerequisites**

* You are a blogger or website
* Free signup to the http://thinktag.co.uk  console

**Features**

* Create your theme in 5 mins
* Add and remove participants
* Download participant list

[See how it works](http://thinktag.co.uk/static/wordpress.html )

== Installation ==

1. Upload the entire 'linktheme' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin using 'Plugins' menu in WordPress.
3. Add the short code to your page or post
